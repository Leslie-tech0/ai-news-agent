> 默认时区：北京时间（UTC+8）

## 附：生成规则

### 必须遵守
- 默认中文输出
- 公司名、模型名、产品名保留原文
- 正文开头不要额外输出标题、生成时间、覆盖时间、版本或来源文件信息，页面标题统一由导出模板生成
- 先提炼趋势，再复盘事件
- 不把日报逐条拼接成周报
- 不做低质量新闻堆砌
- 不写空泛总结
- 强调“为什么重要”和“后续怎么看”

### 用户偏好
- 用户身份：AI 内容创作者 + 投资视角跟踪者
- 用户目标：快速了解本周最重要的 AI 变化，并服务于投资判断与内容选题
- 用户重点关注：
  - 大模型发布 / 更新
  - Agent
  - 编码 AI
  - 多模态
  - AI 硬件 / AI 眼镜 / 机器人
  - GPU / 算力 / 数据中心
  - 投融资
  - AI 应用出海
- 用户不喜欢：
  - 小融资新闻
  - 海外创业公司八卦
  - 泛科技新闻
  - 没有实际落地的 demo
  - 社媒口水战

### 默认优先级
1. OpenAI / Anthropic / Google / NVIDIA / Meta / xAI / Microsoft
2. 大模型与核心产品更新
3. Agent / 编码 AI / 多模态
4. GPU / 算力 / 数据中心
5. 重要融资 / 并购 / API 定价变化
6. AI 应用出海
7. 政策监管
8. 社区热议但未证实的重要信息（仅限高热度，且必须标注）

### 输出原则
- 适合 10-20 分钟阅读
- 信息密度高
- 结构清晰
- 兼顾行业分析感和内容创作价值
- 没有内容的栏目直接省略，不输出占位说明
- 让用户读完后，知道：
  - 这周发生了什么
  - 为什么重要
  - 下周该看什么
  - 哪些能发展成内容题材

- Markdown：`output/weekly/ai-weekly-review-{{week_end}}.md`
- HTML：`output/weekly/ai-weekly-review-{{week_end}}.html`
- PDF：`output/weekly/ai-weekly-review-{{week_end}}.pdf`
